LAB 2 - Esqueleto do site


Entrega: A entrega dessa atividade deve ser um link do github.

Objetivo: Utilizar CSS nas páginas criadas.


HTML: Reproduza duas páginas HTML (‘index.html’, ‘projects.html’):
(valor 1.5) - Todas as páginas devem conter menu (com links) que permita a navegação de uma página para outra.                                  
(valor 1.5) - Todas as páginas devem conter um título utilizando a tag <h1>.
(valor 1.0) - A página ‘Project.html’ deve conter uma tabela com uma linha e duas colunas (“Nome do Projeto” e “Link do projeto”).
Obs: Iremos compor essa tabela ao longo do curso.                  

CSS: Desta vez, se preocupe realmente com a estética das páginas!
(valor 3.0) - Utilize regras de css com seletores de classe, id e tag.        
(valor 1.0) - As regras devem ser definidas em um arquivo externo com a extensão .css  (linkado).                                                 
(valor 2.0) - Utilize pseudo-classes: como o hover, por exemplo.                 


Obs: Outras penalidades podem ser aplicadas.
